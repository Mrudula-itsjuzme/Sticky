# Sticky Roadmap

This roadmap describes planned improvements for the Sticky sticky note application.

## Current focus

- Improve UI polish and note window styling
- Add a title bar or handle for dragging notes more easily
- Persist window position and restore exact note location on launch
- Improve text block resizing and editing UX
- Add a preferences/settings dialog

## Short-term improvements

- [ ] Add richer note editing controls (bold, italic, lists)
- [ ] Implement note title support
- [ ] Add keyboard shortcuts for note actions
- [ ] Add save/load backup support
- [ ] Add support for multiple note themes

## Medium-term improvements

- [ ] Add search across notes and blocks
- [ ] Add note pinning/organization features
- [ ] Add export/import data options
- [ ] Improve desktop integration and portal handling
- [ ] Add a collapsing note/minimize option

## Long-term vision

- [ ] Support cross-device sync
- [ ] Add collaborative editing support
- [ ] Add mobile-friendly version
- [ ] Add plugin/extensions support

## Contribution ideas

If you want to contribute, start with these areas:

- Fixing edge cases for note creation and deletion
- Improving persistence and migration logic in `src/db.rs`
- Adding unit and integration tests for data handling
- Improving styling and theming in `style.css`
- Adding more complete portal support in `src/portals.rs`
