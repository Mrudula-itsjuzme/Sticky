# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-05-28
### Added
- Initial public release of Sticky!
- Floating sticky notes on the desktop.
- Infinite whiteboard canvas view.
- Block-based editing (text, code, timers, checklists).
- Mind-map connections via bezier curves.
- AI meeting summaries (requires OpenAI API key).
- Cross-platform local SQLite storage.
- AppImage and `.deb` packaging for Linux, `.msi` packaging for Windows.
