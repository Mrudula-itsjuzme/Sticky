# Security Policy

## Supported Versions

Currently, only the latest release is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Sticky, please **do not** open a public issue.

Instead, please open a private security advisory on GitHub or email the maintainer directly. We will address all security vulnerabilities promptly.
